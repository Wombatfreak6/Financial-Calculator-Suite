# Financial-Calculator-Suite
FinCalc — Minimal Financial Calculator SPA

A lightweight scroll-based SPA offering clean and simple financial tools — EMI, Savings, Retirement, and Tax calculations — built with vanilla HTML, CSS, and JavaScript.

✦ Features

Smooth scroll navigation between sections

EMI, Compound Interest, Retirement Corpus, Tax calculators

Simple glass UI with responsive design

Auto-highlight active section

Dark mode toggle

Zero dependencies

✦ Project Structure
index.html   # Layout + sections
style.css    # UI styles (glassmorphism + responsive)
script.js    # Logic for calculators + navigation

✦ Usage

Just open:

index.html


Everything runs locally — no build steps, no installations.

✦ Core Calculators

EMI using standard amortization formula

Compound interest growth

SIP-style retirement corpus estimation

Progressive tax with cess across regimes

✦ Tech

HTML + CSS + JavaScript
(no frameworks, libraries, or tooling)
